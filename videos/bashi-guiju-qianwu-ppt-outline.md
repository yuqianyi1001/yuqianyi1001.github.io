## 一、前五识 速览（2 页）

### P4 · 章节扉页
- 标题：一、前五识 速览
- 副标题：眼识 · 耳识 · 鼻识 · 舌识 · 身识

### P5 · 前五识 ↔ 百法
- 「前五识」= 百法 心法 八识里的 前五个
- 体性：以 **了别境界** 为业——眼识了色，耳识了声，鼻识了香，舌识了味，身识了触
- 所缘：**色 / 声 / 香 / 味 / 触** 五尘（五境）
- 所依：**净色根** —— 五根（百法 色法 11 之首）
  - 严格说 = 极微细净色（不是肉团根）
  - 现代角度可理解为 **整个感官机制**——比如 视觉 = 眼球 + 视神经 等的总和
- 在三界中起：欲界 + 色界初禅（详后）

<div class="bg-amber-50 p-3 mt-3">
关键术语区分——五识 有「<strong>了别</strong>」但无「<strong>分别</strong>」：
<ul>
<li><strong>了别</strong>（vijñāna）= 八识本身的功能——直接<strong>辨识</strong>所缘境（红 / 绿 / 高音 / 低音 / 香味 等）。五识都有这种能力，所以叫「识」</li>
<li><strong>分别</strong>（vikalpa）= 在了别之上 加 <strong>名言 / 概念 / 推理 / 判断</strong>——「这是红的」「这是花」「这朵花真美」。这是<strong>第六意识</strong>的功能</li>
</ul>
五识只「了别」不「分别」——眼识看到红色 但不知道「这是花」；判断「这是花」要 第六识 介入。
</div>

## 二、凡夫位（10 页）—— 颂 1 + 颂 2

### P6 · 章节扉页
- 标题：二、凡夫位 · 前五识的运作
- 副标题：颂 1 + 颂 2 · 性境 / 所依 / 心所相应

### P7 · 第一颂 全文 + 大意
> **「性境现量通三性，眼耳身三二地居，遍行别境善十一，中二大八贪嗔痴。」**

- 第一颂讲：**所缘性质 / 起处 / 相应心所 / 相应烦恼**
- 4 句 4 个维度，把前五识的「凡夫位特征」 4 个层面交代清楚

### P8 · 第一颂 · 第一句详解：性境现量通三性

| 五识 | 体性 |
|---|---|
| **境** | <strong>性境</strong> |
| **量** | <strong>现量</strong> |
| **性** | <strong>通三性</strong> |

**三境** —— 一切识所缘的境，按性质分三类：
- **性境**——真实存在的境，被识直接感知；五识所缘
- **带质境**——以真实境为本，但被主观「带上一层概念」的境；比如 第七识缘第八识见分，把它执为「我」
- **独影境**——完全是心识独自变现的影像，没有本质对应；比如 第六识缘的梦境、想象、空花

→ **五识 = 性境**：所缘是真实色法本身，不带概念也不是想象

**三量** —— 认识活动的判断方式，按是否符合事实分三类：
- **现量**——当下直接感知，无名言分别；五识 + 部分第六识
- **比量**——通过推理得到的正确认识；第六识
- **非量**——错误的认识（既不是直接感知，也不是有效推理）；第六识 / 第七识

→ **五识 = 现量**：当下直接感知，不做推理也无判断

**通三性** —— 三性 = 善 / 恶 / 无记：
- 五识自体不分别，三性是与同时俱起的「**心所配伍**」决定
- 同时由 **第六意识引发**——第六识缘善法 → 五识也随善起；第六起恶 → 五识也随染

### P9 · 第一颂 · 第二句详解：眼耳身三二地居
- 「三」= 眼识 + 耳识 + 身识，这 3 个可以在 **欲界 + 色界初禅** 两地起
- 鼻识 + 舌识 = 只在 **欲界** 起。色界众生 已离段食欲（不需物质食物 / 香味），所以无鼻舌识
- 「**二地**」= 五趣杂居地（欲界）+ 离生喜乐地（色界初禅）
  - 按《瑜伽师地论》卷 3 是「**有寻有伺地**」
- 二禅以上 = 无寻唯伺 / 无寻无伺 地，5 识全不起；全靠定中第六识缘内境

### P10 · 第一颂 · 第三句详解：遍行别境善十一
- 与前五识相应的 **善 / 中性心所** 共 21 个：
  - **遍行 5**：触 / 作意 / 受 / 想 / 思
  - **别境 5**：欲 / 胜解 / 念 / 定 / 慧
  - **善 11**：信 / 精进 / 惭 / 愧 / 无贪 / 无嗔 / 无痴 / 轻安 / 不放逸 / 行舍 / 不害

### P11 · 第一颂 · 第四句详解：中二大八贪嗔痴
- 与前五识相应的 **染污心所** 共 13 个：
  - **中随烦恼 2**：无惭 / 无愧
  - **大随烦恼 8**：掉举 / 昏沉 / 不信 / 懈怠 / 放逸 / 失念 / 散乱 / 不正知
  - **根本烦恼 3**：贪 / 嗔 / 痴
- 注 1：小随 10（忿 / 恨 / 覆 / 恼 / 嫉 / 悭 / 诳 / 谄 / 害 / 憍）**不与前五识相应**——《成唯识论》卷 5：「**行相猛利**，唯各别起，前五不与」
- 注 2：根本烦恼里的 **慢 / 疑 / 不正见** 也不与五识相应——需要分别判断，五识 无分别

### P12 · 凡夫位 五识 相应心所 总计
- 21（善/中性）+ 13（染污）= **34 个心所**
- 但具体到某一刹那的五识，不会同时和 34 个相应——根据当时的善 / 恶 / 无记 三性配伍

### P13 · 第二颂 全文 + 大意
> **「五识同依净色根，九缘八七好相邻，合三离二观尘世，愚者难分识与根。」**

- 第二颂讲：**所依根 / 所需缘 / 取境方式 / 凡夫认识误区**

### P14 · 第二颂 · 第一句详解：五识同依净色根
- 「净色根」= 五根的体性 = 极微细净色，**不是肉团**
- 肉团根（眼球、耳廓等）只是「扶尘根」，是净色根的载体
- 五识 所依 = 净色根；不是扶尘根

<div class="bg-stone-50 p-3 mt-3 text-base">
现代理解 ——「净色根」可以理解为<strong>整个感官机制</strong>，而不只是表层的肉眼 / 耳朵。比如「眼根」≈ 眼球 + 视网膜 + 视神经 + 视觉中枢 这一整套；缺一不能见。古人没有解剖学概念，但「净色根」涵盖的是同一类东西。
</div>

### P15 · 第二颂 · 第二句详解：九缘八七好相邻
- 五识 生起需要的缘，数字相邻递减——「好相邻」就是赞这种规律
- **眼识 = 9 缘**：
  1. **空** — 空间距离
  2. **明** — 光明
  3. **根** — 净色根
  4. **境** — 所缘境（色 / 声 / 香 / 味 / 触）
  5. **作意** — 作意心所引发
  6. **分别依** — 第六意识（俱有依）
  7. **染净依** — 第七末那识（俱有依）
  8. **根本依** — 第八阿赖耶识（俱有依）
  9. **种子** — 自识种子（亲因缘）
- **耳识 = 8 缘**：去「明」（黑暗中能闻声）
- **鼻 / 舌 / 身识 = 7 缘**：去「空 + 明」（合中取，不需空间和光明）

<div class="bg-amber-50 p-3 mt-3">
唯识特色：「<strong>分别依 / 染净依 / 根本依</strong>」= 第六 / 七 / 八识 —— 五识 必须 <strong>同时</strong> 有 六 / 七 / 八 识作为「俱有依」才能生起。这是 唯识 八识论 的精髓——所有识 都是<strong>相互依止</strong> 共同运作。
</div>

### P16 · 第二颂 · 第三句详解：合三离二观尘世
- **离中取**：眼识 / 耳识——所缘境 与 根 之间要有空间距离（看远处、听远声）
- **合中知**：鼻识 / 舌识 / 身识——所缘境 必须 与 根接触（鼻闻香气接触鼻根、舌尝味接触舌根、身触接触身根）

### P17 · 第二颂 · 第四句详解：愚者难分识与根
- 凡夫 「愚者」 容易把 **识** 跟 **根** 混淆——
  - 以为「眼睛能看」、其实是「眼识 + 眼根 + 色境 + 等缘」 合力 才能见
  - 单凭 眼根 (净色根) 不能见；单凭 眼识 也不能见
- 这就是 **百法体系 把 心法（八识）与 色法（五根）严格区分** 的意义

### P18 · 凡夫位小结
- 五识 = **现量感知性境**，通三性
- 起处：欲界全有，色界初禅只有眼耳身识
- 相应心所：21 善/中性 + 13 染污 = 共 34 个
- 所依：净色根；所需缘 9 / 8 / 7
- 凡夫 易混 识根

## 三、圣者位（6 页）—— 颂 3

### P19 · 章节扉页
- 标题：三、圣者位 · 前五识的转依
- 副标题：颂 3 · 转为 成所作智

### P20 · 第三颂 全文 + 大意
> **「变相观空唯后得，果中犹自不诠真，圆明初发成无漏，三类分身息苦轮。」**

- 第三颂讲：**五识转依 4 个层面**——所观对象 / 体性 / 转依时机 / 转依果用

### P21 · 第三颂 · 第一句详解：变相观空唯后得
- **变相观空**：五识不能直接缘真如，必须 **变现真如的「相分」** 来观——这是 **后得智**（后得无分别智）的功能
- **唯后得**：五识 不能起 **根本无分别智**（亲证真如），只能起 后得智——因为 五识 不能脱开 相分

### P22 · 第三颂 · 第二句详解：果中犹自不诠真
- 「果」= **佛果**
- 即使到了佛果位，五识 转为 **成所作智** 之后，**也只能现量见相分**，不能「直接诠表」真如
- 因为 五识 是「现量」的——它见 相分（真如的影像），不是「能诠」真如的体性
- 真正 能诠真如 的 = 第六识 妙观察智

### P23 · 第三颂 · 第三句详解：圆明初发成无漏
- **圆明初发**——指 五识 第一次「圆满明朗」地无漏现起，就在 **佛果一刹那**
- 五识 转无漏 **不在见道**——这是 五识 的特点
- 必须等 第八识 转 大圆镜智 之后，五识 才有无漏的所依

### P24 · 第三颂 · 第四句详解：三类分身息苦轮
- **三类分身**——佛果 成所作智 起化用，化现三类身度众生：
  - **大化身**：千丈卢舍那身，为初地以上菩萨化现
  - **小化身**：丈六比丘相，为二乘 / 凡夫 化现
  - **随类化身**：随六道众生 类别 化现各种身形
- **息苦轮**：度脱六道众生 出离苦海

### P25 · 圣者 4 个阶位 · 前五识转依进度（关键页）
| 阶位 | 五识转依进度 |
|---|---|
| **见道（菩萨初地入心）** | 五识 **未转**——仍是有漏。但同时 第六识 / 第七识 已分得 妙观察智 / 平等性智（部分） |
| **修道 二地至七地** | 五识 **仍未转**——但凡夫位 烦恼障的分别起部分已断 |
| **第八不动地** | 五识 **仍未转**——但俱生我执 已伏 |
| **佛果 / 究竟位** | 五识 **与第八识同时** 转 **成所作智**——「**圆明初发**」此刻 |

<div class="bg-stone-50 p-3 mt-3">
严格说 五识 转无漏 <strong>与第八识同时</strong>（《成唯识论》卷 10）——都在 <strong>佛果一刹那</strong> 圆明现起。<br>
不是 五识「最晚」，而是它<strong>不能独立转</strong>——必须等第八识转大圆镜智后，五识 才有 无漏所依，「圆明初发」。
</div>

<div class="bg-amber-50 p-3 mt-2 text-base">
关于「<strong>金刚道</strong>」——是 菩萨 修行的最后一刹那 心（即将成佛的那一念），也就是 <strong>「成佛前的一刹那」</strong>。下一刹那就是佛果。所以本表 把 金刚道 与 佛果 合并表示——一般不单独列阶位。
</div>

| 4 智 | 初转时机 | 圆满时机 |
|---|---|---|
| 妙观察智（第六） | **见道**（初地入心） | 佛果 |
| 平等性智（第七） | **见道**（初地入心，部分）| **八地不动地**（染污末那舍藏）+ 究竟位 完成 |
| 大圆镜智（第八） | **佛果**（成佛一刹那） | 同时即圆满 |
| 成所作智（前五） | **佛果**（与大圆镜智同时） | 同时即圆满 |

### P26 · 圣者位 小结
- 五识转 **成所作智** —— 起 后得智、变相观空
- 转依 **与第八识同时**——佛果一刹那 圆明现起
- 果用 = 三类分身度众生

## 四、综合（3 页）

### P27 · 章节扉页
- 标题：四、凡圣 对照
- 副标题：五识 全貌

### P28 · 凡圣 对照表（核心总结页）
| 维度 | 凡夫位 | 圣者位（佛果） |
|---|---|---|
| **所缘** | 性境（真色法） | 真如的相分 |
| **量** | 现量（直接感知） | 后得智的现量 |
| **性** | 通三性（善恶无记） | 唯善（无漏） |
| **相应心所** | 21 善 / 中性 + 13 染污 = 34 | 只与 21 善 / 中性 相应；染污面全断 |
| **所依** | 有漏净色根 | 无漏净色根 |
| **体性** | 有漏识 | **成所作智** |
| **果用** | 流转生死 | 三类分身息苦轮 |

### P29 · 五识 转依时序图
- **见道（初地入心）** → 修道（二地至七地）→ **不动地（八地）** → **佛果（4 智圆明）**
- 4 智 初转时机：
  - 第六 → 妙观察智：**见道** 初转
