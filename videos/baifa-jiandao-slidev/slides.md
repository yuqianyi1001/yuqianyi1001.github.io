# 见道 ↔ 百法 · 心所变化

<div class="space-y-2 mt-3">

- <strong>顿断 分别起 烦恼</strong>——见解层面的执著
- 百法心所开始<strong>「无漏化」</strong>：
  - <strong>慧</strong>（别境）→ <strong>无漏慧</strong>
  - <strong>念 / 定</strong>（别境）→ 无漏念 / 无漏定
  - <strong>信 / 精进 / 轻安 / 行舍</strong>（善）→ 无漏化
- <strong>七觉支 一时具足</strong>，<strong>八正道 现起</strong>
- 戒体：法处所摄色里的<strong>无表色</strong>形成「圣者戒体」——无漏种子留存

# 88 见所断 · 三界四谛分配 + 与百法对照

<div class="text-base mt-2">

**俱舍说见所断 = 88 个**（这是「具体形态数」，<strong>不是百法心所数</strong>）

| 界 / 谛 | 苦谛 | 集谛 | 灭谛 | 道谛 | 小计 |
| --- | :-: | :-: | :-: | :-: | :-: |
| <strong>欲界</strong> | 10 | 7 | 7 | 8 | <strong>32</strong> |
| <strong>色界</strong>（去嗔）| 9 | 6 | 6 | 7 | <strong>28</strong> |
| <strong>无色界</strong>（去嗔）| 9 | 6 | 6 | 7 | <strong>28</strong> |
| **合计** | | | | | <strong>88</strong> |

</div>

<div class="mt-3 p-3 bg-stone-50 rounded text-base">
<strong>欲界苦谛下 10 个 = </strong>贪 / 嗔 / 痴 / 慢 / 疑 <strong>5 个根本烦恼</strong> + <strong>身见 / 边见 / 邪见 / 见取 / 戒禁取</strong> 5 个见——共 10。<br>
集 / 灭谛各 7 = 去掉 身见 / 边见 / 戒禁取（不缘苦境）。道谛 8 = 加回戒禁取。
</div>

<div class="mt-3 p-3 bg-amber-50 rounded text-base">
<strong>对照百法</strong>：88 个的根本心所其实只有<strong>10 个</strong>——百法<strong>根本烦恼 6 个</strong>（贪 / 嗔 / 痴 / 慢 / 疑 / <strong>不正见</strong>）里的「不正见」拆成 5 见。<br>
《成唯识论》<strong>112 个</strong> = 俱舍 88 + <strong>法执起</strong>的分别烦恼（约 24 个）——唯识双断「我执 + 法执」。
</div>

# 初果须陀洹 · 入流

<div class="space-y-3 mt-3">

- 俱舍说：见道 <strong>第 16 心</strong>（道类智）= 修道第一刹那 = <strong>初果须陀洹</strong>——<strong>预流</strong>（入圣流）

# 阿罗汉的「转依」 · 仅断我执 法执仍在

<div class="text-base space-y-2 mt-2">

- <strong>染污末那</strong>（第七识 4 个俱生我执：<strong>我痴 / 我见 / 我慢 / 我爱</strong>）<strong>断</strong>
  - 第七识 转 <strong>无漏</strong>——但未到「<strong>平等性智</strong>」（菩萨初地以上事）
- <strong>第八识</strong> 舍「阿赖耶」名（不再储存我执种子），但仍是 <strong>异熟识</strong>
- 前五识 / 第六识——保留，但未转 <strong>成所作智 / 妙观察智</strong>（佛果转依）

</div>

阿罗汉时 舍藏；要到 <strong>佛果金刚道</strong> 后 异熟才空。

# 烦恼障 vs 所知障 ↔ 百法

<div class="text-base">

| 障 | 起因 | 百法表现 | 谁能断 |
| --- | --- | --- | --- |
| <strong>烦恼障</strong> | <strong>我执</strong>起的烦恼 | 根本 6 + 随 20（缘我执）| <strong>阿罗汉断</strong> |
| <strong>所知障</strong> | <strong>法执</strong>起的烦恼 | 根本 6 + 随 20（缘法执）| <strong>唯佛能断尽</strong> |

</div>

<div class="mt-5 p-4 bg-stone-50 rounded">
唯识立场——每一个根本烦恼（贪 / 嗔 / 痴 / 慢 / 疑 / 不正见）和随烦恼，<strong>都分「缘我」与「缘法」两版</strong>：
<br>
阿罗汉只断<strong>缘我</strong>那一版；<strong>缘法</strong>那一版要到菩萨道渐断、<strong>佛果断尽</strong>。
</div>

<div class="mt-3 p-3 bg-amber-50 rounded text-base">
所以阿罗汉 ≠ 究竟 ——大乘 / 唯识 立场：<strong>断二障 = 佛果</strong>。

# 心所净化 · 三阶段

<div class="space-y-3 mt-3">

- <strong>资粮 / 加行位</strong>　相似修——「信 / 精进 / 念 / 定 / 慧」<strong>还是有漏</strong>，但方向对
