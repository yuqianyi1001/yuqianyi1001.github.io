# 加行位 · 暖 / 顶 / 忍 / 世第一法

<div class="space-y-2 mt-3">

- <strong>暖</strong>　明得定——智火即将燃起的暖相
- <strong>顶</strong>　明增定——动善根中最高
- <strong>忍</strong>　印顺定——对真如理 心不再退转
- <strong>世第一法</strong>　无间定——世间法中第一，下一刹那入见道
- 四加行 都还是<strong>有漏</strong>——但是凡夫修到极致的状态

</div>

# 加行位 ↔ 百法

- 核心心所：<strong>信 / 精进 / 念 / 定 / 慧</strong>——五根 / 五力 的体性
- 配合：<strong>善法欲</strong>（欲）+ <strong>不放逸</strong>（善）+ <strong>行舍</strong>（善）+ <strong>轻安</strong>（善）

<div class="mt-5 p-4 bg-amber-50 rounded">
<strong>加行位 = 用 信 / 精进 / 念 / 定 / 慧 五个心所，逼近见道</strong>。<br>
体性仍是有漏，但方向已对——下一刹那 就将「质变」为无漏。

# 见道 · 现观四谛十六行相 · 16 心模式

<div class="text-base mt-3">

声闻乘见道——亲见四谛真理，分 <strong>16 个心刹那</strong>（《俱舍论》卷 23）：

- <strong>4 谛 × 4 行相 = 16 行相</strong>
- 每一谛 4 心：法忍 / 法智（缘欲界）+ 类忍 / 类智（缘上二界）
- 共 <strong>8 忍 + 8 智 = 16 心</strong>

</div>

<div class="mt-4 p-3 bg-amber-50 rounded text-base">
《俱舍论》卷 23：「<strong>前 15 心见道，第 16 心修道</strong>」<br>
也就是 8 忍 + 7 智 = 15 心 完成见道；<strong>第 16 心（道类智）= 修道第一刹那 = 初果须陀洹</strong>
</div>

<div class="mt-3 text-base opacity-70">
四谛十六行相 ——苦：无常 / 苦 / 空 / 无我 ；集：因 / 集 / 生 / 缘 ；灭：灭 / 静 / 妙 / 离 ；道：道 / 如 / 行 / 出
</div>

---
layout: default
---

# 见道 ↔ 百法 · 心所变化

<div class="space-y-2 mt-3">

- <strong>顿断 分别起 烦恼</strong>——见解层面的执著
- 百法心所开始<strong>「无漏化」</strong>：
  - <strong>慧</strong>（别境）→ <strong>无漏慧</strong>
  - <strong>念 / 定</strong>（别境）→ 无漏念 / 无漏定
  - <strong>信 / 精进 / 轻安 / 行舍</strong>（善）→ 无漏化
- <strong>七觉支 一时具足</strong>，<strong>八正道 现起</strong>
- 戒体：法处所摄色里的<strong>无表色</strong>形成「圣者戒体」——无漏种子留存

</div>

<div class="mt-5 p-4 bg-amber-50 rounded">
注意：「见道顿断分别起烦恼」是说<strong>这些根本心所在「三界 × 四谛」下的具体形态被断除</strong>——下页详细展开<strong>断的是 88 个 什么</strong>。
</div>

---
layout: default
---

# 88 见所断 · 三界四谛分配 + 与百法对照

<div class="text-base mt-2">

**俱舍说见所断 = 88 个**（这是「具体形态数」，<strong>不是百法心所数</strong>）

| 界 / 谛 | 苦谛 | 集谛 | 灭谛 | 道谛 | 小计 |
| --- | :-: | :-: | :-: | :-: | :-: |
| <strong>欲界</strong> | 10 | 7 | 7 | 8 | <strong>32</strong> |
| <strong>色界</strong>（去嗔）| 9 | 6 | 6 | 7 | <strong>28</strong> |
| <strong>无色界</strong>（去嗔）| 9 | 6 | 6 | 7 | <strong>28</strong> |
| **合计** | | | | | <strong>88</strong> |

</div>

<div class="mt-3 p-3 bg-stone-50 rounded text-base">
<strong>欲界苦谛下 10 个 = </strong>贪 / 嗔 / 痴 / 慢 / 疑 <strong>5 个根本烦恼</strong> + <strong>身见 / 边见 / 邪见 / 见取 / 戒禁取</strong> 5 个见——共 10。<br>
集 / 灭谛各 7 = 去掉 身见 / 边见 / 戒禁取（不缘苦境）。道谛 8 = 加回戒禁取。
</div>

<div class="mt-3 p-3 bg-amber-50 rounded text-base">
<strong>对照百法</strong>：88 个的根本心所其实只有<strong>10 个</strong>——百法<strong>根本烦恼 6 个</strong>（贪 / 嗔 / 痴 / 慢 / 疑 / <strong>不正见</strong>）里的「不正见」拆成 5 见。
</div>

# 初果须陀洹 · 入流

<div class="space-y-3 mt-3">

- 见道 <strong>第 16 心</strong>（道类智）= 修道第一刹那 = <strong>初果须陀洹</strong>——<strong>预流</strong>（入圣流）
- 断 <strong>三结</strong>：
  - <strong>身见</strong>（萨迦耶见）——执五蕴和合为「我」
  - <strong>戒禁取见</strong>——执非戒为戒（持牛狗戒等外道无义戒）
  - <strong>疑</strong>——对四谛 / 三宝 的疑

</div>

<div class="mt-4 p-3 bg-stone-50 rounded text-base">
初果圣者 ——<strong>七番生死内必证阿罗汉</strong>，已入圣道，不堕恶趣。
</div>

# 初果断什么 ↔ 百法

<div class="text-base">

| 百法心所 | 见道时 |
| --- | --- |
| 根本烦恼之 <strong>不正见</strong>（身见 / 边见 / 邪见 / 见取 / 戒禁取）| <strong>分别起 · 顿断</strong> |
| 根本烦恼之 <strong>疑</strong> | <strong>断</strong> |
| 根本烦恼之 <strong>贪 / 嗔 / 痴 / 慢</strong> | 见所断者断；<strong>俱生起者留</strong> |
| 随烦恼 20 | 见所断者断；俱生起者留 |
| <strong>善心所 11 / 别境 5 / 遍行 5</strong> | <strong>全保留 + 部分无漏化</strong> |

</div>

<div class="mt-4 p-3 bg-amber-50 rounded">
初果 = <strong>「见」的层面已破</strong>——但情感惯性（俱生烦恼）还在，要靠修道渐断。
</div>

# 修道 = 渐断 9 地 81 品烦恼

<div class="space-y-3 mt-3">

- <strong>见所断</strong> = 「分别起」的烦恼——见解层面，<strong>一断顿断</strong>
- <strong>修所断</strong> = 「俱生起」的烦恼——情感惯性，<strong>要慢慢断</strong>
- <strong>9 地</strong>：欲界 + 4 个根本静虑 + 4 个无色界 = 九地
- <strong>9 品</strong>：上上 / 上中 / 上下 / 中上 / 中中 / 中下 / 下上 / 下中 / 下下
- 9 × 9 = <strong>81 品</strong> 修所断烦恼

</div>

<div class="mt-5 p-3 bg-stone-50 rounded text-base">
修所断的核心 = 根本烦恼里的 <strong>贪 / 嗔 / 痴 / 慢</strong> 四个 俱生起部分。
</div>

---
layout: default

# 二果斯陀含 · 三果阿那含

<div class="space-y-2 mt-3 text-base">

- <strong>二果向</strong>　断欲界 修所断 第 1-5 品贪嗔痴慢
- <strong>二果斯陀含（一来）</strong>　断欲界 第 6 品——还要回欲界一次
- <strong>三果向</strong>　断欲界 第 7-8 品
- <strong>三果阿那含（不还）</strong>　断欲界 全 9 品——<strong>不再回欲界</strong>，生色界 / 无色界 继续修

</div>

<div class="mt-4 p-4 bg-amber-50 rounded">
用百法看——<strong>根本烦恼里的「贪 / 嗔 / 痴 / 慢」俱生起部分被逐步剥离</strong>。<br>
善 / 别境 / 遍行 心所 全保留，并逐步无漏化。
</div>

# 四果阿罗汉 · 烦恼断尽

- 断尽 上二界（色 / 无色）修所断 <strong>72 品</strong>烦恼
- 三界 见所断 + 修所断 <strong>全断</strong>——「<strong>漏尽</strong>」
- 自知「<strong>我生已尽，梵行已立，所作已办，不受后有</strong>」
- 称「<strong>无学</strong>」——三无漏学（戒 / 定 / 慧）圆满，再无应学（《俱舍论》卷 24）

<div class="mt-5 p-4 bg-stone-50 rounded">
但这只是 <strong>声闻乘的极果</strong>——用大乘 / 唯识标准看，还<strong>未到究竟</strong>。
</div>

# 阿罗汉 断什么 ↔ 百法

<div class="text-base">

| 百法 | 阿罗汉时 |
| --- | --- |
| <strong>根本烦恼 6</strong>（贪 / 嗔 / 痴 / 慢 / 疑 / 不正见）| <strong>全断</strong> |
| <strong>随烦恼 20</strong>（忿 / 恨 / 覆 / 恼 / 嫉 / 悭 等）| <strong>全断</strong> |
| 不定心所 4 中的<strong>染污面</strong>（恶作 / 睡眠 / 寻 / 伺 的不善部分）| <strong>断</strong> |

</div>

<div class="mt-5 p-3 bg-amber-50 rounded">
阿罗汉 = 三界 见所断 + 修所断 的<strong>染污心所 全部断尽</strong>——「漏尽」。<br>
善心所、别境、遍行 等 染污面 之外的心所 都保留并无漏化（见下页）。
</div>

# 阿罗汉 保留什么 ↔ 百法

<div class="text-base">

| 百法 | 阿罗汉时 |
| --- | --- |
| <strong>善心所 11</strong> | <strong>全保留 + 全无漏化</strong> |
| <strong>别境心所 5</strong>（欲 / 胜解 / 念 / 定 / 慧）| <strong>保留 + 无漏化</strong>（特别是「慧」转为无漏慧）|
| <strong>遍行心所 5</strong>（触 / 作意 / 受 / 想 / 思）| 保留（无漏识相应）|
| 不定心所 4 中的 善 / 无记 面 | 保留 |
| <strong>心王 8 识</strong> | 保留（百法立 8 识；声闻乘只立 6 识，此处不展开）|
| 色法 11 / 不相应行 24 中的 善 / 中性法 | 保留 |

# 阿罗汉 vs 佛果 · 大乘观点

<div class="space-y-3 mt-3">

- 声闻乘 看——阿罗汉 = <strong>极果</strong>。三界烦恼断尽，「不受后有」，永不再来。
- 大乘 / 唯识 看——<strong>阿罗汉 ≠ 究竟</strong>：还有「法执」、还有更深的「所知障」要断。
- <strong>佛果</strong> = 二障俱断 + 转 八识 为 四智 —— 才是真正的究竟。

</div>

<div class="mt-5 p-4 bg-amber-50 rounded">
本讲 主线 = 声闻乘的 见道 → 阿罗汉。<strong>菩萨道 / 佛果 的更深转依</strong>（所知障 / 八识转四智 / 不动地前才舍藏 等）属于 唯识 大乘体系，本讲<strong>不展开</strong>。
</div>

<div class="mt-3 text-base opacity-70">
注：声闻乘 体系（如《俱舍论》）不立 所知障 / 法执，也不立 第七识 / 第八识；这些都是大乘 / 唯识 独有的概念。
</div>

# 心所净化 · 三阶段

<div class="space-y-3 mt-3">

- <strong>资粮 / 加行位</strong>　相似修——「信 / 精进 / 念 / 定 / 慧」<strong>还是有漏</strong>，但方向对
- <strong>见道位</strong>　一刹那「质变」——这几个心所从有漏转<strong>无漏</strong>；顿断 <strong>88 个</strong>分别起 见所断烦恼
- <strong>修道 → 无学位</strong>　渐断 <strong>9 地 81 品</strong>俱生烦恼——阿罗汉位 三界<strong>烦恼断尽</strong>

</div>

<div class="mt-5 p-4 bg-amber-50 rounded">
百法视角下的修证——<strong>心所没有被「消灭」，是被「净化」</strong>：染污面断，善 / 别境 / 遍行 面保留并无漏化。
</div>

<div class="mt-3 p-3 bg-stone-50 rounded text-base">
阿罗汉 = 声闻乘 极果；<strong>大乘观点 — 佛果 才是究竟</strong>。本讲不展开。
</div>

# 全篇回顾 · 五讲系列总结

<div class="space-y-2 mt-3 text-base">
