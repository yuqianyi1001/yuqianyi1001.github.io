## 三、见道（3 页）

### P9 · 章节扉页
- 标题：三、见道
- 副标题：凡夫到圣者的「分水岭」

### P10 · 见道 · 真见道 + 相见道（唯识立场）
- **唯识把见道分两层**（《成唯识论》卷 9）——
  - **真见道**　一刹那 · **根本无分别智 现见 二空真如**（人空 + 法空）——这是见道的「本体」
  - **相见道**　真见道后 后得智 安立四谛 → **16 心**（依俱舍说）或 **3 心**（唯识细分）——这是见道的「相」（认知层面）
- 四谛 × 四行相 = 十六行相（与俱舍说相同）
  - 苦谛：无常 / 苦 / 空 / 无我
  - 集谛：因 / 集 / 生 / 缘
  - 灭谛：灭 / 静 / 妙 / 离
  - 道谛：道 / 如 / 行 / 出
- **对照**：《俱舍论》把 16 心都视为「见道」（8 忍 + 8 智）；《成唯识论》把「真见道」拈出，强调 **现见真如 才是本质**

### P11 · 见道 ↔ 百法
- 顿断 **分别起**的烦恼——见解层面的执著
- **数目**（唯识立场）：
  - 《俱舍论》: 见所断 **88 个**
  - 《成唯识论》: 见所断 **112 个**——比俱舍多 24 个，因为唯识把**法执分别起**的烦恼也算见所断（俱舍只算我执起的）
- **唯识双断 我执 + 法执**——这是与俱舍的关键不同
- 见道时——**七觉支一时具足**、**八正道现起**
- 百法心所开始「无漏化」：慧 转为无漏慧，念 / 定 转为无漏念 / 定
- 戒体：无表色（法处所摄色）的无漏种子留存——「圣者戒体」

## 四、初果须陀洹（2 页）

### P12 · 初果须陀洹 · 入流
- 见道第 16 心 = 道类智 = **初果须陀洹**——预流（入圣流）
- 断 **三结**：
  - **身见**（萨迦耶见）——执五蕴和合为「我」
  - **戒禁取见**——执非戒为戒（持牛狗戒等外道无义戒）
  - **疑**——对四谛的疑
- 「七番生死」内必证阿罗汉——已入圣道，不堕恶趣

### P13 · 初果断什么 ↔ 百法
| 百法心所 | 见道断除？ |
| --- | --- |
| 根本烦恼之 **不正见**（身见、边见、邪见、见取、戒禁取）| **顿断分别起** |
| 根本烦恼之 **疑** | **断** |
| 根本烦恼之 **贪、嗔、痴、慢** | 见所断者**断**；俱生起者**留** |
| 随烦恼 20 | 见所断者断；俱生起者留 |
| 善心所 11、别境 5、遍行 5 | **全保留 + 部分无漏化** |

## 五、修道与 二果 / 三果（3 页）

### P14 · 章节扉页
- 标题：五、修道 · 渐断俱生烦恼
- 副标题：从 一来 到 不还

### P15 · 修道 = 渐断 9 地 81 品烦恼
- 见所断 = 「分别起」的烦恼——见解层面，一断顿断
- 修所断 = 「俱生起」的烦恼——情感惯性，要慢慢断
- **9 地**：欲界 + 4 个根本静虑 + 4 个无色界 = 9 地
- **9 品**：上上、上中、上下、中上、中中、中下、下上、下中、下下
- 9 × 9 = **81 品** 修所断烦恼

### P16 · 二果斯陀含（一来）· 三果阿那含（不还）
- **二果向**：断欲界 修所断 第 1-5 品贪嗔痴慢
- **二果斯陀含**（一来）：断欲界 第 6 品——还要回欲界一次
- **三果向**：断欲界 第 7-8 品
- **三果阿那含**（不还）：断欲界 全 9 品——**不再回欲界**，生色界 / 无色界继续修
- 用百法看：**根本烦恼里的「贪 / 嗔 / 痴 / 慢」俱生起部分被逐步剥离**

## 六、四果阿罗汉（4 页）

### P17 · 章节扉页
- 标题：六、四果阿罗汉
- 副标题：烦恼断尽 · 无学位

### P18 · 四果阿罗汉 · 烦恼断尽
- 断尽 上二界（色 / 无色）修所断 **72 品**烦恼
- 三界见所断 + 修所断**全断**——「漏尽」
- 自知「我生已尽，梵行已立，所作已办，不受后有」
- 称「**无学**」——再无可学的有漏法

### P19 · 阿罗汉 断什么 ↔ 百法
| 百法 | 阿罗汉时 |
| --- | --- |
| 根本烦恼 6 | **全断**（贪 / 嗔 / 痴 / 慢 / 疑 / 不正见） |
| 随烦恼 20 | **全断** |
| 不定心所 4 中的 染污面（恶作、睡眠 的不善 / 寻 / 伺 的染污）| **断** |

### P20 · 阿罗汉 保留什么 ↔ 百法
| 百法 | 阿罗汉时 |
| --- | --- |
| 善心所 11 | **全保留 + 全无漏化** |
| 别境心所 5（欲 / 胜解 / 念 / 定 / 慧）| **保留 + 无漏化**（特别是「慧」转为无漏慧）|
| 遍行心所 5（触 / 作意 / 受 / 想 / 思）| **保留**（无漏识相应）|
| 不定心所 4 中的 善 / 无记 面 | **保留** |
| 心王 8 识 | **保留**——但**第七识染污末那断**，前六识、第八识仍存（详见下页）|
| 色法 11、不相应行 24 中的 善法 / 中性法 | **保留** |

### P21 · 阿罗汉的「转依」 · 仅断我执，法执仍在
> 这一页是唯识 vs 俱舍 最大差异——必看。

- **染污末那**（第七识的 4 个俱生我执：**我痴 / 我见 / 我慢 / 我爱**）**断**
  - 第七识转 **无漏**——但未到「**平等性智**」（那是菩萨初地以上事）
- **第八识** 舍「阿赖耶」名（不再储存我执种子），但仍是 **异熟识**——
  - 《八识规矩颂》：「**不动地前才舍藏，金刚道后异熟空**」
  - 阿罗汉时 舍藏；要到 **佛果金刚道** 后 异熟才空
- 前五识 / 第六识——保留，但未转**成所作智 / 妙观察智**（佛果转依）
- **核心判教**（《成唯识论》卷 9）：
  - 阿罗汉只断 **烦恼障**（我执起的烦恼）
  - 法执还在 → **所知障未断** → 仍有「法执」（执法实有）
  - 菩萨道目标 = 断**二障** = 佛果

### P22 · 烦恼障 vs 所知障 ↔ 百法
| 障 | 起因 | 数量 | 谁能断 |
| --- | --- | --- | --- |
| **烦恼障** | **我执**起的烦恼 | 根本 6 + 随 20（缘我执） | 阿罗汉断 |
| **所知障** | **法执**起的烦恼 | 根本 6 + 随 20（缘法执） | **唯佛能断尽** |

> 唯识立场：每一个根本烦恼（贪、嗔、痴、慢、疑、不正见）和随烦恼（如忿、恨、嫉等）都分**缘我**与**缘法**两版——阿罗汉只断缘我那一版，缘法的那一版到菩萨道才渐断、佛果断尽。

## 关键校验点（按《成唯识论》/《八识规矩颂》/《瑜伽师地论》）

1. **见道 = 真见道 + 相见道**——《成唯识论》卷 9：「**真见道**根本无分别智，亲证二空真如；**相见道**后得智，安立四谛」
2. **见所断 112 烦恼**——唯识立场含**法执**分别起的烦恼（俱舍说 88，仅算我执起）
3. **修所断 81 品**——9 地 × 9 品（与俱舍共许）
4. **四加行配四寻思 / 四如实智**——《成唯识论》卷 9
5. **染污末那 4 烦恼**——我痴 / 我见 / 我慢 / 我爱（《成唯识论》卷 5）
6. **「不动地前才舍藏，金刚道后异熟空」**——《八识规矩颂》第八识：阿罗汉舍藏，佛果空异熟
7. **烦恼障 vs 所知障**——《成唯识论》卷 9：阿罗汉断烦恼障 / 菩萨断所知障；二障俱断 = 佛果
8. **八识转四智**——《八识规矩颂》：前五 → 成所作智；六 → 妙观察智；七 → 平等性智；八 → 大圆镜智
9. **菩萨五位**——资粮 / 加行 / 通达 / 修习 / 究竟（《成唯识论》卷 9）
10. **法执细分**：分别起法执 = 见所断；俱生法执 = 修所断（《成唯识论》卷 9）
