# 三十七道品 各组的功能定位

<div class="space-y-2 mt-3">

- <strong>四念住</strong>　入门 · <strong>观</strong>身、受、心、法，破四颠倒
- <strong>四正勤</strong>　精进发力 · <strong>断恶修善</strong>
- <strong>四神足</strong>　得定 · 由欲 / 勤 / 心 / 观 引发<strong>禅定</strong>
- <strong>五根 → 五力</strong>　信进念定慧 · 从<strong>能生善法</strong>转向<strong>能伏烦恼</strong>
- <strong>七觉支</strong>　修道位 · 圣者于见道后增长的<strong>七种觉悟支分</strong>
- <strong>八正道</strong>　见道现起 + 修道继续 · 圣者修证之总路

</div>

<div class="mt-5 p-3 bg-amber-50 rounded text-base">
这几组不是孤立的——是<strong>修证渐进的几个阶段</strong>。
</div>

# 有漏 / 无漏 的关键差别

<div class="text-base">

| 道品 | 凡夫位修 | 圣者位修 |
| --- | --- | --- |
| 四念住 / 四正勤 / 四神足 | <strong>有漏</strong>（相似修） | 见道后转<strong>无漏</strong> |
| 五根 / 五力 | <strong>有漏</strong>（暖顶位）| 忍世第一以上转<strong>无漏</strong> |
| <strong>七觉支 · 八正道</strong> | 只是<strong>相似训练</strong> | <strong>唯是无漏</strong>——真实道谛 |

</div>

<div class="mt-5 p-4 bg-stone-50 rounded">
《俱舍论》：「<strong>七觉八道支，一向是无漏</strong>」
</div>

<div class="mt-3 text-base opacity-70">

佛教鼓励凡夫修"相似八正道"，是趋向真实道的训练。

# 「修证」的本质

<div class="space-y-3 mt-4">

- <strong>三十七道品 不是三十七个独立的修法</strong>
- 而是上面这 <strong>八个核心心所</strong>——
  - 在不同<strong>阶段</strong>（念住 → 正勤 → 神足 → 根 → 力 → 觉支 → 道支）
  - 用不同<strong>组合</strong>反复训练
  - 让它们从<strong>有漏</strong>逐步转为<strong>无漏</strong>

</div>

<div class="mt-6 p-4 bg-amber-50 rounded">
<strong>修行的根本工作</strong> = 训练这几个善心所，让它们<strong>坚固 · 清净 · 有力</strong>。
</div>

# 四念住 ↔ 百法

- 核心心所：<strong>念</strong>（别境心所之一）+ <strong>慧</strong>（别境心所之一）
- 配合：<strong>不放逸</strong>（善心所之一）+ <strong>思</strong>（遍行心所之一）+ <strong>作意</strong>（遍行心所之一）

<div class="mt-4 p-4 bg-stone-50 rounded text-base">
所观对象 = 百法本身的内容——<br>
身念住 → 色法 11（五根 / 五尘 / 法处所摄色）<br>
受念住 → 受心所<br>
心念住 → 心王 8 识<br>
法念住 → 想心所 + 行蕴其他心所 + 不相应行 24 + 无为法 6
</div>

<div class="mt-4 p-3 bg-amber-50 rounded">
<strong>四念住 = 拿「念 + 慧」两个心所，去观察整个百法</strong>。

# 凡夫位修的是「相似七觉支」

- 凡夫位的 <strong>念 / 慧 / 精进 / 喜 / 轻安 / 定 / 舍</strong> 都是<strong>有漏</strong>

<div class="mt-5 p-4 bg-stone-50 rounded">
