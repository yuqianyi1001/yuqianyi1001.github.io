### 完整描述（公众号 / B 站正文）

《大乘百法明门论》系列**第四讲**——与 **三十七道品** 对照，
把百法落实到修证路径上。

#### 本期核心论断

> **三十七道品 不是三十七条独立的修法——
> 而是反复在练几个核心善心所和别境心所。**

#### 七组三十七道品 一图对应

- **四念住**（4）—— 身念住 · 受念住 · 心念住 · 法念住　观破四颠倒（净 / 乐 / 常 / 我）
- **四正勤**（4）—— 已生恶令断 / 未生恶令不生 / 未生善令生 / 已生善令增
- **四神足**（4）—— 欲神足 · 勤神足 · 心神足 · 观神足　得定的四种助伴
- **五根**（5）—— 信根 · 精进根 · 念根 · 定根 · 慧根　从「能生善法」立名
- **五力**（5）—— 信力 · 精进力 · 念力 · 定力 · 慧力　从「能伏烦恼」立名
- **七觉支**（7）—— 念 / 择法 / 精进 / 喜 / 轻安 / 定 / 舍 觉支　修道位 唯无漏
- **八正道**（8）—— 正见 / 正思惟 / 正语 / 正业 / 正命 / 正精进 / 正念 / 正定　见道现起 + 修道继续

#### 几个 反常识 的校验点

- 七觉支 不是"觉悟的资粮"——是**修道位**圣者于见道后增长的觉悟支分；
- 八正道 也不是凡夫日常守的"八条道德戒律"——是圣者**见道现起 + 修道继续**的总路；
- 凡夫修的"正见"等只是名字相似（"相似八正道"），不是真实的八正道；
- **五根 / 五力** 不是力量大小的两个阶段——是同一套 信进念定慧，
  从「**能生善法**」（根 · 生善）与「**能伏烦恼**」（力 · 伏恶）两个角度立的名；
- **戒体** ≠ 戒条——正语 / 正业 / 正命 对应百法里"法处所摄色"中的**无表色**，
  是身语二业留存下来的善种子。

#### 经典依据

本讲严格按《俱舍论》《成唯识论》《八识规矩颂》校验——
- 四念住「以慧为自体，因念力相续，故名念住」　《俱舍论》卷 23
- 五根 / 五力「能生为根，能伏为力」　《大智度论》＋《俱舍论》卷 25
- 七觉八道支「一向是无漏」　《俱舍论》卷 25
- 八正道里 身语二业 由第六识 + 思心所驱动　《八识规矩颂》「动身发语独为最」

#### 本期核心論斷

> **三十七道品 不是三十七條獨立的修法——
> 而是反復在練幾個核心善心所和別境心所。**

#### 七組三十七道品 一圖對應

- **四念住**（4）—— 身念住 · 受念住 · 心念住 · 法念住　觀破四顛倒（淨 / 樂 / 常 / 我）
