## 一、三十七道品 速览（4 页）

### P4 · 章节扉页
- 一、三十七道品 速览
- 副标：声闻乘修证的总纲

### P5 · 七组 三十七道品
- **四念住**　4　身念住、受念住、心念住、法念住
- **四正勤**（四正断）　4　已恶令断 / 未恶令不生 / 未善令生 / 已善令增
- **四神足**（四如意足）　4　欲神足 / 勤神足 / 心神足 / 观神足
- **五根**　5　信根 / 进根 / 念根 / 定根 / 慧根
- **五力**　5　信力 / 进力 / 念力 / 定力 / 慧力
- **七觉支**（七等觉支）　7　念/择法/精进/喜/轻安/定/舍
- **八正道**（八圣道支）　8　正见/正思惟/正语/正业/正命/正精进/正念/正定
- 合计 4+4+4+5+5+7+8 = **37**

### P6 · 七组各自的功能定位
- **四念住** ——**入门** · 观身、受、心、法，破四颠倒
- **四正勤** ——**精进发力** · 断恶修善
- **四神足** ——**得定** · 由欲勤心观引发禅定
- **五根 → 五力** ——**信进念定慧** · 从嫩芽长成成熟力量
- **七觉支** ——**修道位** · 圣者于见道后增长的七种觉悟支分
- **八正道** ——**见道现起 + 修道继续** · 圣者修证之总路

### P7 · 有漏 / 无漏 的关键差别
| 道品 | 凡夫位修 | 圣者位修 |
| --- | --- | --- |
| 四念住 / 四正勤 / 四神足 / 五根 / 五力 | **有漏**（相似修） | 见道后转**无漏** |
| **七觉支 · 八正道** | 只是**相似训练** | **唯是无漏**——真实道谛 |
- 《俱舍论》：**「七觉八道支，一向是无漏」**
- 凡夫的"正见"等只是名字相似，**不是真实的八正道**
- 佛教鼓励凡夫修"相似八正道"——是趋向真实道的训练

## 二、修证 = 强化几个善心所（核心论点）（3 页）

### P8 · 章节扉页
- 二、三十七道品 的"百法真相"
- 副标：**重复使用几个善心所**

### P9 · 八个核心心所 撑起 37 个道品
| 心所 | 百法分类 | 在三十七道品中出现的位置 |
| --- | --- | --- |
| **信** | 善 11 | 信根、信力 |
| **精进** | 善 11 | 四正勤、精进根/力、精进觉支、正精进 |
| **念** | 别境 5 | 四念住、念根/力、念觉支、正念 |
| **定** | 别境 5 | 四神足（心神足）、定根/力、定觉支、正定 |
| **慧** | 别境 5 | 慧根/力、择法觉支、正见（含正思惟） |
| **轻安** | 善 11 | 轻安觉支 |
| **行舍** | 善 11 | 舍觉支 |
| **欲**（善法欲）| 别境 5 | 四正勤"愿"、四神足（欲神足）|

### P10 · 「修证」的本质
- 三十七道品**不是 37 个独立的修法**
- 而是上面这 **8 个善心所**——
  - 在不同**阶段**（念住→正勤→神足→根→力→觉支→道支）
  - 用不同**组合**反复训练
  - 让它们从**有漏**逐步转为**无漏**
- **修行的根本工作 = 训练这几个善心所，让它们坚固、清净、强力**

## 三、四念住（2 页）

### P11 · 四念住 · 总览
| 念住 | 观察对象 | 行相 | 破颠倒 |
| --- | --- | --- | --- |
| 身念住 | 自身色身 | **观身不净** | 破"净"倒 |
| 受念住 | 苦/乐/舍三受 | **观受是苦** | 破"乐"倒 |
| 心念住 | 六识心王 | **观心无常** | 破"常"倒 |
| 法念住 | 想行蕴 + 无为法 | **观法无我** | 破"我"倒 |

### P12 · 四念住 ↔ 百法
- 核心心所：**念**（别境）+ **慧**（别境）
- 配合：**不放逸**（善）+ **思**（遍行）+ **作意**（遍行）
- 所观对象 = 百法本身的内容：
  - 身念住 → 色法 11 中的五根 / 五尘 / 法处所摄色
  - 受念住 → 受心所
  - 心念住 → 心王（主要指第 6 识：意识）
  - 法念住 → 想心所 + 行蕴的其他 心所 + 不相应行 + 无为法
- **结论**：四念住 = **拿"念" + "慧"两个心所，去观察整个百法**

## 四、四正勤（2 页）

### P13 · 四正勤 · 总览
- **已生恶**　令断
- **未生恶**　令不生
- **未生善**　令生
- **已生善**　令增广
- 主体是"精进"——所以也叫四正勤

### P14 · 四正勤 ↔ 百法
- 核心心所：**精进**（善）
- 配合：**善法欲**（欲，别境）+ **思**（遍行）+ **不放逸**（善）+ **行舍**（善）
- 内容上对应：
  - 所断的"恶" = 根本烦恼 6 + 随烦恼 20 + 不定中的染污面
  - 所修的"善" = 善 11 心所
- **结论**：四正勤 = **拿"精进 + 善法欲"两个心所，去推动断恶修善**

---

## 五、四神足（2 页）

### P15 · 四神足 · 总览
- 别名：四如意足
- 体性 = **定**
- 四种助伴让定生起：
  - **欲神足**　善法欲推动定
  - **勤神足**　不懈精进推动定
  - **心神足**　心一境性专注推动定
  - **观神足**　如理思惟推动定

### P16 · 四神足 ↔ 百法
- 核心心所：**定**（别境）
- 四个助伴 ↔ 百法心所：
  - 欲 → **欲**（别境）
  - 勤 → **精进**（善）
  - 心 → **定**（别境，专注一境）
  - 观 → **慧**（别境）
- **结论**：四神足 = **拿"欲 / 精进 / 念定 / 慧"四个心所，把禅定推上来**

## 六、五根 → 五力（2 页）

### P17 · 五根 = 五力 = 信进念定慧
- **五根**：能生善法的"根基"——如树根，能长出枝叶花果
- **五力**：能破烦恼的"力量"——同样是信进念定慧，但势力强盛
- 关键差别：
  - **根**：能动摇，遇逆缘可退
  - **力**：不退、不被烦恼摧伏
- 暖顶位 → 五根；忍世第一位以上 → 五力

### P18 · 五根五力 ↔ 百法
- 直接对应百法 **5 个心所**：
  - **信**（善 11 之一）
  - **精进**（善 11 之一）
  - **念**（别境 5 之一）
  - **定**（别境 5 之一）
  - **慧**（别境 5 之一）
- **结论**：五根 / 五力 = **同一套 5 个心所，强弱不同的两个阶段**

## 七、七觉支（3 页）

### P19 · 七觉支 · 总览
- 全称：七等觉支 / 七菩提分
- **念　/ 择法　/ 精进　/ 喜　/ 轻安　/ 定　/ 舍**
- 《俱舍论》：「七觉八道支，**一向是无漏**」
- **唯是无漏**——见道以上才真实成就

### P20 · 七觉支 ↔ 百法
| 觉支 | 百法心所 |
| --- | --- |
| **念觉支** | 念（别境） |
| **择法觉支** | 慧（别境） |
| **精进觉支** | 精进（善） |
| **喜觉支** | 受（遍行，乐受）+ 信 / 轻安 的合力 ——「法喜」|
| **轻安觉支** | 轻安（善） |
| **定觉支** | 定（别境） |
| **舍觉支** | 行舍（善） |

### P21 · 凡夫位修的是"相似七觉支"
- 凡夫位的 **念 / 慧 / 精进 / 喜 / 轻安 / 定 / 舍** 都是**有漏**
- 同名异实：用同样的名字标示，但体性、所依、所缘、果报完全不同
- 凡夫位的训练是**通向真实觉支的预备功课**
- 见道时，七觉支**一时具足**，共缘四谛实相

## 八、八正道（3 页）

### P22 · 八正道 · 总览
- 正见 / 正思惟 / 正语 / 正业 / 正命 / 正精进 / 正念 / 正定
- 见道后**圆满展开**的修证总路
- 同样**唯是无漏**——凡夫修的是"相似八正道"

### P23 · 八正道 ↔ 百法
| 八支 | 百法心所 / 业 |
| --- | --- |
| **正见** | 慧（别境，无漏慧） |
| **正思惟** | 思（遍行）+ 慧 |
| **正语** | 不妄语等 → **语业**（思心所驱动）|
| **正业** | **身业**（不杀盗淫，由思心所驱动）|
| **正命** | 身业 + 语业 + 不贪等 |
| **正精进** | 精进（善） |
| **正念** | 念（别境） |
| **正定** | 定（别境） |

> **戒体层面**：正语 / 正业 / 正命 同时对应法处所摄色中的「**无表色**」——身语二业不仅是当下行为，还会形成一种"无表"的善种子留存。

### P24 · 八正道 的"百法骨架"
- **3 个别境心所 + 1 个善心所 + 3 业**
  - 别境：念、定、慧
  - 善：精进
  - 三业：身、语、命（由第六识 + 思心所发动）
- 也就是说：八正道 = **同样几个心所，在身语意三业上全面落实**
- 这就是为什么佛说"**八正道支独尊**"——它涵盖了修证的所有维度
