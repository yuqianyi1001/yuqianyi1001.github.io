# 三科速览

- <strong>五蕴</strong>　5：色、受、想、行、识
- <strong>十二处</strong>　12：六根 + 六尘
- <strong>十八界</strong>　18：六根 + 六尘 + 六识

<div class="mt-6 p-4 bg-amber-50 rounded">
关键：<strong>三科 = 同一套法的三种切片</strong>，不是三套不同的法。
</div>

<div class="mt-4 text-base opacity-70">
小提示：五蕴是一切<strong>有为法</strong>。「一切法」有时只指有为法（如《金刚经》「一切有为法，如梦幻泡影」），有时兼指有为 + 无为。
</div>

<!--
三科 等于 五蕴 加 十二处 加 十八界——是佛教对一切法的三种基础归纳。
五蕴 5 个：色、受、想、行、识。
十二处 12 个：六根 加 六尘。
十八界 18 个：六根、六尘、再加六识。
最关键的一点——三科不是三套不同的法，而是同一套法的三种切片。
另外补一句小知识：五蕴是一切有为法。「一切法」在《金刚经》里有时只指有为法，有时兼指有为加无为，要看上下文。
-->

# 五蕴 ——「是什么」的角度

- <strong>色</strong>　物质（身体 + 外境）
- <strong>受</strong>　感受（苦 / 乐 / 舍）
- <strong>想</strong>　概念、贴标签
- <strong>行</strong>　造作、意志
- <strong>识</strong>　了别

<div class="mt-5 p-4 bg-stone-50 rounded text-base">
<strong>「做饭」比喻</strong>（《俱舍论颂疏》圆晖 + 智敏上师《讲记》扩说）——
色蕴 = 厨房厨具　·　受蕴 = 食材　·　想蕴 = 调料　·　行蕴 = 大厨　·　识蕴 = 食客
</div>

<!--
五蕴回答的是「这一切法是什么」。
色：物质，包括身体和外境。
受：感受，分苦受、乐受、舍受。
想：概念，给事物贴标签。
行：造作，意志活动。
识：了别，认识本身。
《俱舍论颂疏》（唐代圆晖）有一个"做饭"比喻，智敏上师《讲记》里又详细扩说——
色蕴是厨房和厨具；受蕴是食材；想蕴是调料；行蕴是大厨；识蕴是食客。
五个加在一起，才能完成一场认识活动。
-->

# 十八界 ——「完整认识链」

<div class="space-y-3 mt-4">

- <strong>百法</strong>：11 + 1 + 1 + 73 + 8 = <strong class="text-amber-700">94</strong>，差 <strong>6</strong> 个无为法 → 100

# 为什么无为法不入蕴

- <strong>「蕴」的本义</strong> = 积聚、聚集
- 只有<strong>有为法</strong>才能聚集——增减、起灭、和合
- 无为法（虚空、择灭、非择灭、真如…）<strong>无生灭、不增不减</strong>，无法用「聚集」形容

<div class="mt-6 text-base opacity-70">
《大毗婆沙论》用大量篇幅对比"蕴"与"无为"，结论一致——所以五蕴不摄无为法。
</div>

<!--

# 受、想 为什么单独立蕴

- 心所有 51 个（俱舍 46），为什么独独把<strong>「受」和「想」</strong>单立为蕴？
- 《俱舍论颂疏讲记》原文：<strong>「受与想，于诸行中，相粗，生染」</strong>

<div class="mt-4 space-y-2">

- <strong>最粗</strong>（仅次于色蕴）
- <strong>能引生染污</strong>：乐受 → 贪；苦受 → 嗔；想 → 起名分别 → 染净
- <strong>二界中强</strong>：色界中受蕴强，无色界中想蕴强

</div>

<div class="mt-4 p-3 bg-stone-50 rounded text-base">
食材和调料缺一不可：没有它们，色蕴的厨房 + 行蕴的大厨，也做不出菜给识蕴的食客。
</div>

<!--

# 意根 / 意界 · 两种说法

- <strong>百法</strong>：意根 = <strong>第七识（末那识）</strong>
- <strong>俱舍 75 法</strong>：意根 = <strong>前一刹那的意识</strong>
- 《俱舍论颂疏讲记》：「意识了别之后的下一刹那，就变成了意根。」

<div class="mt-5 p-4 bg-amber-50 rounded text-base">
<strong>怎么看？</strong>当成知识点记住即可——阿罗汉若不闻佛说，也不知有第七、第八识；也就无法区分意根是"第七"还是"前一刹那意识"。<br>
普通人记住一种说法就够用。
</div>

<div class="mt-3 text-base opacity-70">
本系列沿用 <strong>百法的解释</strong>——意根 = 末那识。
</div>
