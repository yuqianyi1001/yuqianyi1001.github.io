# 什么是「三性」

任何心识活动，按其性质，必落入三种之一——

- <strong>善</strong>　清净 · 利他 · 感招可爱果
- <strong>恶</strong>（不善）　染污 · 损害 · 感招苦果
- <strong>无记</strong>　非善非恶 · 再细分两类：
  - <strong>有覆无记</strong>：本身不造善恶业，但<strong>被烦恼覆障</strong>真理
    <span class="opacity-70 text-base">　典型：第七识的「我执」</span>
  - <strong>无覆无记</strong>：纯然中立，<strong>不覆障</strong>真理
    <span class="opacity-70 text-base">　典型：第八识的执持</span>

<!--
三性，是把任何心识活动，按其性质，归为三类。
第一类，善——清净、利他，感招可爱果。
第二类，恶（也叫不善），染污、损害，感招苦果。
第三类，无记——非善非恶。
但无记还要再细分两类——
有覆无记：本身不直接造善恶业，但被烦恼覆障，遮蔽真理。第七识就是典型。
无覆无记：纯然中立，不覆障真理。第八识就是典型。
这三性，是评判每一个识、每一个心所「性格」的最基本框架。
-->

# 前五识的三性

- <strong>通三性</strong>：善、恶、无记都可以
- 但其中无记部分都是<strong>「无覆无记」</strong>
- 前五识本身只是<strong>「现量取性境」</strong>——不分别善恶
- 它的三性，<strong>随俱起的第六识（五俱意识）而转</strong>

<div class="mt-6 p-4 bg-stone-50 rounded">
意识起信 → 五识同<strong>善</strong>；意识起贪嗔 → 五识同<strong>恶</strong>；意识中性观看 → 五识也是<strong>无覆无记</strong>。
</div>

<div class="mt-3 text-base opacity-70">
依据：《八识规矩颂》「性境现量通三性」
</div>

# 第六识的三性

- <strong>三性全通</strong>——善、恶、无记，且无记中<strong>有覆与无覆都有</strong>

- 它是最<strong>自由</strong>、也最<strong>危险</strong>的一识

# 第七识的三性

# 「相应」到底是什么意思

不是"协调一致"，而是唯识里的专门术语——

<div class="mt-3 grid grid-cols-2 gap-4">
  <div class="border rounded p-4">
    <div class="font-bold">同一刹那</div>
    <div class="opacity-70 text-base mt-1">在同一个时间点共起</div>
  </div>
  <div class="border rounded p-4">
    <div class="font-bold">同一所依</div>
    <div class="opacity-70 text-base mt-1">依止同一个识</div>
  </div>
  <div class="border rounded p-4">
    <div class="font-bold">同一所缘</div>
    <div class="opacity-70 text-base mt-1">观看同一个对象</div>
  </div>
  <div class="border rounded p-4">
    <div class="font-bold">同一事</div>
    <div class="opacity-70 text-base mt-1">完成同一件功能</div>
  </div>
</div>

<div class="mt-4 p-3 bg-amber-50 rounded text-base">
凡是和某识能在同一刹那共起的心所，就叫与该识「<b>相应</b>」。
</div>

<!--
现在解释一个非常重要的概念——「相应」。
唯识里说的相应，不是「协调一致」那种泛义，而是一个有四层精确含义的专门术语——
同一刹那：在同一个时间点共起。
同一所依：依止同一个识。
同一所缘：观看同一个对象。
同一事：完成同一件功能。
凡是和某识能在同一刹那共起、共缘、共事的心所，就叫与该识「相应」。
有了这个定义，我们就可以一个识一个识地看——它分别和哪些心所相应。
-->

# 为什么这些会相应

前五识本身只是<strong>「现量取性境」</strong>——它的相应，是<strong>随俱起的第六识同起同缘</strong>而成立的。

- <strong>遍行 5</strong>：心王活动的基础，前五识自身必有
- <strong>别境 5</strong>：与俱意识同缘所观境时一并现起
- <strong>善 11</strong>：俱意识起善心所时，前五识随同
- <strong>贪 / 嗔 / 痴</strong>：粗的烦恼，现量层面就能与俱意识同起
- <strong>无惭 / 无愧</strong>：俱意识起恶心必带，前五识也不例外
- <strong>大随 8</strong>：俱意识起染污心必带，前五识普遍相伴

<div class="mt-4 p-3 bg-amber-50 rounded text-base">
关键：<strong>不是前五识自己分别善恶，而是它随俱意识同起同缘</strong>。
</div>
