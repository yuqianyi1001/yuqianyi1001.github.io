# 为什么要学《八识规矩颂》

- 《百法》体系里，**八识** = <strong>心法 / 心王</strong>——位居 五位之首
- 玄奘大师 造的《八识规矩颂》四章十二颂 —— 是唯识 / 法相宗 学习八识 最经典的入门论
- 《百法》给的是「**地图**」（八识是哪 8 个、有什么心所）
- 《八识规矩颂》给的是「**运作 + 转依**」（每个识在凡夫位怎么运作、在圣者位怎么转）

<div class="mt-5 p-4 bg-amber-50 rounded">
<strong>学习前提</strong>：对《百法》中的 <strong>51 个心所 + 它们的分类</strong> 都要了解——遍行 5 / 别境 5 / 善 11 / 根本烦恼 6 / 随烦恼 20 / 不定 4。本讲会快速复习，但不重新展开每个心所的含义。
</div>

# 前五识 ↔ 百法

<div class="text-lg space-y-2 mt-3">

- 「前五识」= 百法 心法八识里的前五个
- 体性：<strong>了别境界</strong>——眼了色 / 耳了声 / 鼻了香 / 舌了味 / 身了触
- 所缘：<strong>色 / 声 / 香 / 味 / 触</strong> 五尘
- 所依：<strong>净色根</strong>（极微细净色 ≈ 整个感官机制）
- 起处：欲界 + 色界初禅

</div>

<div class="mt-4 p-3 bg-amber-50 rounded text-base">
五识有「<strong>了别</strong>」但无「<strong>分别</strong>」——<br>
<strong>了别</strong>＝直接辨识所缘境（红 / 绿 / 香味）；<br>
<strong>分别</strong>＝加名言概念（「这是花」），是第六识的功能。
</div>

# 第一颂 · 第一句详解：性境现量通三性

<div class="mt-2 p-3 bg-amber-50 rounded text-lg">
<strong>白话</strong>：前五识所缘 是 真实色法本身（不是想象，也不是带概念的执取）；当下直接感知（不做推理）；自己不能决定善恶，可以善、可以恶、可以无记。
</div>

<div class="grid grid-cols-2 gap-3 mt-4 text-base">
  <div class="p-3 bg-stone-50 rounded">
    <strong>三境</strong>——按所缘境性质分三类：
    <ul class="mt-1">
      <li><strong>性境</strong>　真实存在，直接感知（五识所缘）</li>
      <li><strong>带质境</strong>　以真实境为本 加一层概念（第七识把第八识见分执为「我」）</li>
      <li><strong>独影境</strong>　心识独自变现的影像（梦境 / 想象 / 空花）</li>
    </ul>
  </div>
  <div class="p-3 bg-stone-50 rounded">
    <strong>三量</strong>——按认识判断方式分三类：
    <ul class="mt-1">
      <li><strong>现量</strong>　当下直接感知（五识 + 部分第六识）</li>
      <li><strong>比量</strong>　通过推理得到的正确认识（第六识）</li>
      <li><strong>非量</strong>　错误的认识（第六识 / 第七识）</li>
    </ul>
  </div>
</div>

<div class="mt-3 p-3 bg-amber-50 rounded text-base">
<strong>通三性</strong>＝善 / 恶 / 无记 三性都可起。五识自体不分别，三性由<strong>心所搭配 + 第六识引发</strong>共同决定。
</div>

# 第一颂 · 第二句详解：眼耳身三二地居

<div class="mt-2 p-3 bg-amber-50 rounded text-lg">
<strong>白话</strong>：眼识、耳识、身识 在欲界 + 色界初禅 两地起；鼻识、舌识 只在欲界起；二禅以上 前五识 全部停。
</div>

<div class="space-y-3 mt-3">

- 「三」= <strong>眼识 + 耳识 + 身识</strong>，这三个可以在欲界 + 色界初禅 起
- <strong>鼻识 + 舌识</strong> = 只在欲界起。色界众生已离段食欲（不需物质食物 / 香味），所以无鼻舌识
- 「<strong>二地</strong>」= 五趣杂居地（欲界）+ 离生喜乐地（色界初禅）
  - 按《瑜伽师地论》卷 3 是「<strong>有寻有伺地</strong>」
- <strong>中间禅</strong>（初二禅之间）= <strong>无寻有伺地</strong>——只有伺，无寻
- 二禅以上 = <strong>无寻无伺地</strong>——寻伺都无，五识全不起；全靠定中第六识缘内境

</div>

# 第一颂 · 第三句详解：遍行别境善十一

<div class="mt-2 p-3 bg-amber-50 rounded text-lg">
<strong>白话</strong>：跟前五识可以相应的 善 / 中性心所，共 21 个——遍行 5 + 别境 5 + 善 11。
</div>

与前五识相应的 <strong>善 / 中性心所</strong> 共 21 个——

<div class="text-base mt-4">

| 类别 | 个数 | 内容 |
| --- | :-: | --- |
| <strong>遍行</strong> | 5 | 触 / 作意 / 受 / 想 / 思 |
| <strong>别境</strong> | 5 | 欲 / 胜解 / 念 / 定 / 慧 |
| <strong>善</strong> | 11 | 信 / 精进 / 惭 / 愧 / 无贪 / 无嗔 / 无痴 / 轻安 / 不放逸 / 行舍 / 不害 |

</div>

<div class="mt-5 p-3 bg-amber-50 rounded text-base">
合计 <strong>21 个善 / 中性心所</strong>——这是五识所有可能的「善 / 中性」相应心所。
</div>

# 第一颂 · 第四句详解：中二大八贪嗔痴

<div class="mt-2 p-3 bg-amber-50 rounded text-lg">
<strong>白话</strong>：跟前五识可以相应的 染污心所，共 13 个——中随 2 + 大随 8 + 根本 3（贪嗔痴）。小随 10 和 慢/疑/不正见 不相应。
</div>

与前五识相应的 <strong>染污心所</strong> 共 13 个——

<div class="text-base mt-3">

| 类别 | 个数 | 内容 |
| --- | :-: | --- |
| <strong>中随烦恼</strong> | 2 | 无惭 / 无愧 |
| <strong>大随烦恼</strong> | 8 | 掉举 / 昏沉 / 不信 / 懈怠 / 放逸 / 失念 / 散乱 / 不正知 |
| <strong>根本烦恼</strong> | 3 | 贪 / 嗔 / 痴 |

</div>

<div class="mt-4 p-3 bg-stone-50 rounded text-base">
<strong>注 1</strong>：小随 10（忿 / 恨 / 覆 / 恼 / 嫉 / 悭 / 诳 / 谄 / 害 / 憍）<strong>不与前五识相应</strong>——《成唯识论》卷 5：「<strong>行相猛利</strong>，唯各别起，前五不与」<br>
<strong>注 2</strong>：根本烦恼里的<strong>慢 / 疑 / 不正见</strong>也不与五识相应——需要分别判断，五识 无分别
</div>

# 第二颂 · 第一句详解：五识同依净色根

<div class="mt-2 p-3 bg-amber-50 rounded text-lg">
<strong>白话</strong>：前五识 都依「净色根」起。净色根 不是肉眼可见的肉团（眼球、耳朵），而是 整个感官机制 的精微部分。
</div>

<div class="space-y-3 mt-3">

- 「净色根」= 五根的体性 = <strong>极微细净色</strong>，不是肉团
- 肉团根（眼球、耳廓等）只是「<strong>扶尘根</strong>」，是净色根的载体
- 五识 所依 = <strong>净色根</strong>；不是扶尘根

</div>

<div class="mt-5 p-3 bg-stone-50 rounded text-base">
现代理解——「净色根」可以理解为<strong>整个感官机制</strong>，而不只是表层的肉眼 / 耳朵。<br>
比如「眼根」≈ 眼球 + 视网膜 + 视神经 + 视觉中枢 这一整套；缺一不能见。古人没有解剖学概念，但「净色根」涵盖的是同一类东西。
</div>

---
layout: default
---

# 第二颂 · 第二句详解：九缘八七好相邻

<div class="mt-2 p-3 bg-amber-50 rounded text-base">
<strong>白话</strong>：前五识 生起 需要的「缘」——眼识 9 个、耳识 8 个、鼻舌身 各 7 个；缺一 不能起。
</div>

<div class="grid grid-cols-2 gap-4 mt-3 text-base">
  <div>
    <strong>眼识 = 9 缘</strong>：
    <ol class="mt-2">
      <li><strong>空</strong>——空间距离</li>
      <li><strong>明</strong>——光明</li>
      <li><strong>根</strong>——净色根</li>
      <li><strong>境</strong>——所缘境</li>
      <li><strong>作意</strong>——作意心所引发</li>
      <li><strong>分别依</strong>——第六意识（俱有依）</li>
      <li><strong>染净依</strong>——第七末那识（俱有依）</li>
      <li><strong>根本依</strong>——第八阿赖耶识（俱有依）</li>
      <li><strong>种子</strong>——自识种子（亲因缘）</li>
    </ol>
  </div>
  <div class="space-y-3">
    <div class="p-3 bg-stone-50 rounded">
      <strong>耳识 = 8 缘</strong>：去「明」<br>
      <span class="opacity-70">黑暗中能闻声</span>
    </div>
    <div class="p-3 bg-stone-50 rounded">
      <strong>鼻 / 舌 / 身识 = 7 缘</strong>：去「空 + 明」<br>
      <span class="opacity-70">合中取，不需空间和光明</span>
    </div>
    <div class="p-3 bg-amber-50 rounded">
      <strong>唯识特色</strong>：「分别依 / 染净依 / 根本依」= 第六 / 七 / 八识。<br>
      五识必须 <strong>同时</strong>有六 / 七 / 八识作为「俱有依」才能生起——所有识 <strong>相互依止</strong> 共同运作。
    </div>
  </div>
</div>

# 第二颂 · 第三句详解：合三离二观尘世

<div class="mt-2 p-3 bg-amber-50 rounded text-lg">
<strong>白话</strong>：前五识 取境分两种——眼识、耳识 要有距离（离中取）；鼻识、舌识、身识 必须接触（合中知）。
</div>

<div class="space-y-3 mt-3">

- <strong>离中取</strong>：眼识 / 耳识——所缘境 与 根 之间要有空间距离（看远处、听远声）
- <strong>合中知</strong>：鼻识 / 舌识 / 身识——所缘境 必须 与 根接触
  - 鼻闻香气 接触 鼻根
  - 舌尝味 接触 舌根
  - 身触 接触 身根

</div>

<div class="mt-5 p-4 bg-amber-50 rounded">
所以「<strong>合三离二</strong>」= 合中知 3 识 + 离中取 2 识 = 五识的取境方式分两类。
</div>

# 第二颂 · 第四句详解：愚者难分识与根

<div class="mt-2 p-3 bg-amber-50 rounded text-lg">
<strong>白话</strong>：凡夫「愚者」 容易 把「识」跟「根」混淆——以为「眼睛能看」，其实 要 多个缘 合力 才能见。
</div>

<div class="space-y-3 mt-3">

- 凡夫「愚者」 容易把 <strong>识</strong> 跟 <strong>根</strong> 混淆——
  - 以为「眼睛能看」
  - 其实是「眼识 + 眼根 + 色境 + 作意 + 种子 + ... 等缘」合力 才能见
- 单凭 眼根（净色根）不能见；单凭 眼识 也不能见

</div>

<div class="mt-5 p-4 bg-amber-50 rounded">
这就是<strong>百法体系 把 心法（八识）与 色法（五根）严格区分</strong> 的意义——它们的体性不同，缘别也不同，不能混为一谈。
</div>

# 凡夫位小结

<div class="space-y-3 mt-4 text-xl">

- 五识 = <strong>现量感知性境</strong>，通三性
- 起处：欲界全有，色界初禅只有眼耳身识
- 相应心所：21 善 / 中性 + 13 染污 = 共 <strong>34 个</strong>
- 所依：净色根；所需缘 9 / 8 / 7
- 凡夫 易混 识 根

</div>

<div class="mt-6 p-4 bg-stone-50 rounded">
凡夫位 五识——<strong>受第六识引发，与众多心所相应搭配，在三界二地中起 现量感知</strong>。但因不知 识根之别，流转生死。
</div>

# 第三颂 · 第一句详解：变相观空唯后得

<div class="mt-2 p-3 bg-amber-50 rounded text-lg">
<strong>白话</strong>：前五识 即使到佛果，能观察 诸法空性，也 不是 直接 亲证 真如——它 必须 先现起 一个「空」的相，再借这个相 来观察。所以 这种观空 属于「后得智」，不是「根本智」。
</div>

<div class="space-y-3 mt-3">

- <strong>变相观空</strong>：五识不能直接缘真如，必须<strong>变现真如的「相分」</strong>来观——这是<strong>后得智</strong>（后得无分别智）的功能
- <strong>唯后得</strong>：五识 不能起<strong>根本无分别智</strong>（亲证真如），只能起 后得智——因为 五识 不能脱开 相分

</div>

<div class="mt-5 p-4 bg-amber-50 rounded">
为什么？——五识 本质是<strong>现量感知性境</strong>，必有「相分」可缘。而根本无分别智 直接亲证 真如，没有相分；这种功能 不是五识 所能。
</div>

---
layout: default
---

# 第三颂 · 第二句详解：果中犹自不诠真

<div class="mt-2 p-3 bg-amber-50 rounded text-lg">
<strong>白话</strong>：即使 到了佛果，前五识 转成 成所作智 之后，也 不能 直接「诠表」真如——它 只能 现量感知 相分，不能 用 名言概念 把 真如 表达出来。
</div>

<div class="space-y-3 mt-3">

- 「果」= <strong>佛果</strong>
- 即使到了佛果位，五识 转为<strong>成所作智</strong>之后，也只能<strong>现量见相分</strong>，不能「直接诠表」真如
- 因为五识 是「现量」的——它见相分（真如的影像），不是「能诠」真如的体性
- 真正能<strong>诠真如</strong>的 = 第六识 妙观察智

</div>

<div class="mt-5 p-4 bg-stone-50 rounded">
所以五识 即使在佛果，也只是<strong>现量见相分</strong>。能跨过相分 直接 诠真 的，唯 妙观察智。
</div>

layout: default
---

# 第三颂 · 第三句详解：圆明初发成无漏

<div class="mt-2 p-3 bg-amber-50 rounded text-lg">
<strong>白话</strong>：前五识 第一次「圆满 明朗」地 起为 无漏，就在 佛果 一刹那（与 第八识 同时转）。在这之前 一直 都是 有漏。
</div>

<div class="space-y-3 mt-3">

- <strong>圆明初发</strong>——指五识 第一次「圆满明朗」地无漏现起，就在 <strong>佛果一刹那</strong>
- 五识 转无漏 <strong>不在见道</strong>——这是五识 的特点
- 必须等 第八识 转 大圆镜智 之后，五识 才有无漏的所依

</div>

<div class="mt-5 p-4 bg-amber-50 rounded">
关键点——五识 不能独立转。<strong>必须等第八识转大圆镜智 之后，五识 才有无漏的所依 ——「圆明初发」</strong>。
</div>

# 第三颂 · 第四句详解：三类分身息苦轮

<div class="mt-2 p-3 bg-amber-50 rounded text-lg">
<strong>白话</strong>：佛果 成所作智 起化用，化现 三类身 度众生——大化身、小化身、随类化身；度脱 六道众生 出离 苦海。
</div>

佛果 成所作智 起化用，化现三类身度众生——

<div class="text-xl mt-4">

| 化身 | 对象 | 形相 |
| --- | --- | --- |
| <strong>大化身</strong> | 初地以上菩萨 | 千丈卢舍那身 |
| <strong>小化身</strong> | 二乘 / 凡夫 | 丈六比丘相（如释迦应身） |
| <strong>随类化身</strong> | 六道众生 | 随其类别 化现各种身形 |

</div>

<div class="mt-5 p-4 bg-amber-50 rounded">
<strong>息苦轮</strong> = 度脱六道众生 出离苦海。<br>
所以 成所作智 的果用 = <strong>三类化身度众</strong>。
</div>
