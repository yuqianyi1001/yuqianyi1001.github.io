# 一切法

佛學中，用“一切法”來概括佛法所要認識的對象，事務。包括一切物質的，一切精神的，也包括一切的理論，抽象，概念，都包括了。佛學中，也經常用“萬法”來代替“一切法”使用。

一切法，最常用的又有這樣幾個分類。

## 有為法、無為法

一切法 = 有為法 + 無為法

一切因緣所生的法，都是有為法。反之，不是因緣所生的法，就是無為法。

## 五蘊

五蘊是：色蘊、受蘊、想蘊、行蘊、識蘊。

不過要注意的是，因為“蘊”本身是聚集的意思，就是有從少到多可以聚集，由微細到眾多可以聚集，所以，五蘊全部都是有為法。所以，五蘊加起來，就是一切的有為法。

一切有為法 = 五蘊 = 色蘊 + 受蘊 + 想蘊 + 行蘊 + 識蘊

## 十二處

十二處是：眼處、耳處、鼻處、舌處、身處、意處、色處、聲處、香處、味處、觸處、法處。  
前面六個又叫內六處：眼處、耳處、鼻處、舌處、身處、意處。  
後面六個又叫外六處：色處、聲處、香處、味處、觸處、法處。

上面的內外分類是依據其中的大多數而言，不是嚴格的定義。因為法處，是包括無為法的。無為法不分內外。

一切法 = 十二處 = 眼處 + 耳處 + 鼻處 + 舌處 + 身處 + 意處 + 色處 + 聲處 + 香處 + 味處 + 觸處 + 法處





但是要注意，佛經中的“一切法”有時會有限定條件，或隱含的限定條件。後面再列舉。
