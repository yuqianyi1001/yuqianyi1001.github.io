# 一切法

佛学中，用“一切法”来概括佛法所要认识的所有对象，事物。包括一切物质的，一切精神的，也包括一切的理论，抽象，概念，都包括了。佛学中，也经常用“万法”来表示一切法”。

一切法，最常用的有这样几个分类：

## 有为法、无为法

一切法 = 有为法 + 无为法

一切因缘所生的法，都是有为法。反之，不是因缘所生的法，就是无为法。

## 五蕴

五蕴是：色蕴、受蕴、想蕴、行蕴、识蕴。

不过要注意的是，因为“蕴”本身是聚集的意思，就是有从少到多可以聚集，由微细到众多可以聚集，所以，五蕴全部都是有为法。所以，五蕴加起来，就是一切的有为法。

一切有为法 = 五蕴 = 色蕴 + 受蕴 + 想蕴 + 行蕴 + 识蕴

## 十二处

十二处是：眼处、耳处、鼻处、舌处、身处、意处、色处、声处、香处、味处、触处、法处。  
前面六个又叫内六处：眼处、耳处、鼻处、舌处、身处、意处。  
后面六个又叫外六处：色处、声处、香处、味处、触处、法处。

上面的内外分类是依据其中的大多数而言，不是严格的定义。因为法处，是包括无为法的。无为法不分内外。

一切法 = 十二处 = 眼处 + 耳处 + 鼻处 + 舌处 + 身处 + 意处 + 色处 + 声处 + 香处 + 味处 + 触处 + 法处

## 十八界

十八界 = 眼界、耳界、鼻界、舌界、身界、意界、色界、声界、香界、味界、触界、法界、眼识界、耳识界、鼻识界、舌识界、身识界、意识界

这里也可以分组，  
前面六个叫做内六界：眼界、耳界、鼻界、舌界、身界、意界。 
中间六个叫做外六界：色界、声界、香界、味界、触界、法界。  
后面六个叫做六识界：眼识界、耳识界、鼻识界、舌识界、身识界、意识界。  

这里也一个法界，对应法处。也包括了无为法。所以，

一切法 = 十八界 = 眼界 + 耳界 + 鼻界 + 舌界 + 身界 + 意界 + 色界 + 声界 + 香界 + 味界 + 触界 + 法界 + 眼识界 + 耳识界 + 鼻识界 + 舌识界 + 身识界 + 意识界

## 综合

一切法 = 十二处 = 十八界 = 俱舍七十五法 = 唯识百法 = 有为法 + 无为法  

一切有为法 = 五蕴

## 佛经中的一切法

但是要注意，佛经中的“一切法”有时会有限定条件，或隐含的限定条件。

例如《金刚经》，“一切有为法，如梦幻泡影，如露亦如电，应作如是观”，这里就是说的一切有为法，不包括无为法了。所以，这里的一切有为法，可以用五蕴来代替，而不能用十二处、十八界来代替了。

当然，佛经中的“一切法”有时也有其他含义，可能是“一切善法”，“一切佛法“等，具体要看上下文了。
