# 藏传格鲁辩经，一次中文演示｜附完整的文字稿和我的评论

藏传佛教有辩经的传统，但是，大多都是用藏文相互辩论，对于不懂藏文的来说，很难有机会观摩并能够听懂。

偶尔间，我在油管上发现了一个中文辩经的视频，而且所辩论的内容，也在我所知的范围之类。于是，我就想要把它记录下来，并且分享一二。

有条件看油管的朋友，可以直接看视频：https://www.youtube.com/watch?v=gDPreuPLlhU （备注，我仅仅关注这里的佛学辩论。）

没条件看油管的朋友，可以听这个音频。

简单介绍下这里辩论的内容，是《现观庄严论》的一个偈颂：

**《現觀莊嚴論略釋》卷1：「發心為利他，求正等菩提，彼彼如經中，略廣門宣說。」(CBETA 2025.R1, B09, no. 31, p. 21a9)**

讨论了两个话题：
1. 佛陀有没有菩提心？
2. 佛陀有没有皈依？

说明：
* 挑战者：即发问的人，提问的人。
* 被挑战者：即需要回答问题的人。

## 文字实录

（因为辩论是口语化的，因此，下面文字实录有略作删减，提高可读性，但是保留95%以上的原文。）

**挑战者：** “发心为利他，求正等菩提”。请问这句话是出自哪一个经典呢？

<span style="color:blue">被挑战者： 《现观庄严论》

**挑战者：** 这句话是出自《现观庄严论》吗？

<span style="color:blue">被挑战者： 是的

**挑战者：** 那如果是出自《现观庄严论》的话，那么这句话是出自《现观庄严论》的第几品呢？

<span style="color:blue">被挑战者： 第一

**挑战者：** 那出自第一品的哪一个段落呢？

<span style="color:blue">被挑战者： 发心的段落

**挑战者：** 好，既然是发心的段落的话，那请问您可以把这整句偈颂完整的念出来吗？

<span style="color:blue">被挑战者： 好

**挑战者：** 请说

<span style="color:blue">被挑战者： “发心为利他，求正等菩提，彼彼如经中，略广门宣说”

**挑战者：** 好，“发心为利他，求正等菩提，彼彼如经中，略广门宣说”，那您说这句话是出自《现观庄严论》的发心的部分，对不对？

<span style="color:blue">被挑战者： 是

**挑战者：** 那请您先依照这句话的经文做解释。在讲出这句话当中，关于发心的定义。

<span style="color:blue">被挑战者： 这个偈颂显示了发心的定义。首先说发心为利他，求正等菩提。说到发心需要具备两种希求，两种欲求，也就是要欲求想要利他，这是第一个欲求。然后第二个求菩提这样的一个欲求，具备这两种欲求的殊胜的心事，它算是菩提心的定义。

**挑战者：** 但后面两句呢？

<span style="color:blue">被挑战者： “彼彼如经中，略广门宣说”，这个“彼彼”分别指广略的般若，广般若，指的是10万般若。略般若，指的是8000颂。“如经中”包含了中般若，也就是两万颂般若。

**挑战者：** 所以“略广门宣说”就是从广中略的般若经中开示了发心的内涵。我帮你收摄一下，“彼彼如经中，略广门宣说”。“彼彼”指的就是般若10万颂以及般若8000颂，如同般若2万颂当中，展示了发心的名相跟定义，这样的一个方式，“略”的开示跟”广“的开示发心的定义。是这样子吗？

<span style="color:blue">被挑战者： 是

**挑战者：** 如果是的话，在“发心为利他，求正等菩提”这两句话当中，同时包含了这个发心菩提心的现象以及定义，对吗？

<span style="color:blue">被挑战者： 是的，

**挑战者：** 那它的定义当中又包含了两个。第一个是智力的欲求以及利他的欲求，对吗？

<span style="color:blue">被挑战者： 是，

**挑战者：** 再来菩提心。我们讲完菩提心的定义之后，菩提心有哪一些分类呢？

<span style="color:blue">被挑战者： 菩提心从体性的角度，可以分为愿心和行心

**挑战者：** 可以分为愿心和行心吗？那有没有其他的分类方式呢？

<span style="color:blue">被挑战者： 从诠说身的角度，能诠身的角度可以分为胜义菩提心和世俗菩提心。

**挑战者：** 那有没有更广的呢？

<span style="color:blue">被挑战者： 更广的话可以分为22种发心。

**挑战者：** 再来，我们讲完了菩提心的分类之后，具有菩提心的行者，他的范围是从哪里到哪里呢？他的界定该如何界定呢？

<span style="color:blue">被挑战者： 从下品的资粮道到一直到果位之间，这个是大乘的，大乘的下品资粮道开始到佛陀（佛地）之间。

**挑战者：** 所有在其中的的补特伽罗心中都一定具有菩提心吗？

<span style="color:blue">被挑战者： 是的，

**挑战者：** 那如果是的话，只要是佛陀的话，那他的心中一定具备菩提心吗？

<span style="color:blue">被挑战者： 是的，

**挑战者：** 如果是这样的话，以释迦牟尼佛为例，对吧？他的心中有菩提心吗？

<span style="color:blue">被挑战者： 有的

**挑战者：** 有吗？

**挑战者：** 以释迦牟尼佛为例，他的心中应该没有菩提心，为什么呢？因为他的心中并不具备自他两种欲求的缘故

**挑战者：** 应不成

**挑战者：** 应该是成立的。因为释迦牟尼佛的心中并没有智力的欲求。

**挑战者：** 为什么没有智力的欲求呢？因为释迦牟尼佛已经自利圆满的缘故，如果已经自利圆满的话，那怎么还会有自利的需求呢？

<span style="color:blue">被挑战者： 应不成

<span style="color:blue">被挑战者： 如果自利圆满，为什么要自利的欲求？他有求菩提的心。

**挑战者：** 好，那如果是这样的话，已经断尽烦恼的阿罗汉来说，他的心中还有想要断烦恼的心吗？

<span style="color:blue">被挑战者： 有

**挑战者：** 有吗，他这个已经断掉，他还想要再断吗？还有更多的可以断吗？

<span style="color:blue">被挑战者： 他自己没有更多要断

**挑战者：** 他自己没有更多要断。那还有谁要断呢？

<span style="color:blue">被挑战者： 还有有烦恼的有情的烦恼要断

**挑战者：** 还有烦恼要断，谁的烦恼？

<span style="color:blue">被挑战者： 其他有情，没有断烦恼的有情，心续当中的烦恼要断。

**挑战者：** 好，那对吗？如果其他有情的心续当中还有烦恼要断，对不对？已经断除烦恼的阿罗汉，他的心续中还有想要断除烦恼的欲求吗？

<span style="color:blue">被挑战者： 是

**挑战者：** 这个欲求您指的是说，他的心中虽然没有想要断除自己的烦恼的欲求，但是有想要断除他人心中烦恼的欲求。是这样子吗？

**挑战者：** 是的，

**挑战者：** 那如果是这样子的话，这阿罗汉断除烦恼的阿罗汉，他的心绪当中的想要断除他人心中的烦恼的这个欲求。以这个欲求为例，他是自利的欲求吗？

<span style="color:blue">被挑战者： 不算自利的欲求

**挑战者：** 他不算自利的欲求，因为他自利，他这个要断的东西已经断干净了，对不对？如果是这样的话，那么佛陀心中的想要欲求菩提的，欲求菩提的这个欲求，那么是不是智力的欲求呢？

<span style="color:blue">被挑战者： 不是

**挑战者：** 不是吗？那如果是这样的话，那佛陀的心中就不具备自利以及利他两种欲求了。

<span style="color:blue">被挑战者： 是的，

**挑战者：** Oh cha。如果是这样的话，那佛陀的心中不具备不菩提心吗？

**挑战者：** 不是

**挑战者：** 因为具备菩提心的话，不需要具备两种欲求的缘故？

**挑战者：** 不遍

**挑战者：** 应该是周遍的。因为佛陀心续当中的菩提心。您说他是菩提心，但是又说他不是两种欲求的缘故，又不具备两种欲求的缘故人，那只要是菩提心的话，不周遍是这个具备两种欲求的发心吗？

<span style="color:blue">被挑战者： 是

**挑战者：** 应该周遍的，因为这个是发心的定义

**挑战者：** 应不成

**挑战者：** cha，对吧？这个矛盾了吧。刚才您不是说在讲到发心的定义当中，你是怎么讲“发心为利他，求正等菩提”的时候，你说“为利他”，他指的是利他的欲求，“求正等菩提”指的是智力的欲求。那只要是发心，那必定具备两种欲求。这不是坐在我面前的这个立宗者，您的根本立宗吗？

<span style="color:blue">被挑战者： 不是

**挑战者：** 那不是的话，那就cha。如果是这样的话，那么这个“发心为利他，求正等菩提”的偈颂当中，他所开示的，并不是这个发心的定义吗？

<span style="color:blue">被挑战者： 是发心的定义。

**挑战者：** 如果是的话，你要怎么解释说这个“为利他”跟“求正等菩提”这两种欲求呢？因为刚才有提到说，只要是菩提心的话，并不一定具备这两种欲求吗？好，那怎么解释？

<span style="color:blue">被挑战者： 字面上所讲到的发心定义是指有学位行者发心的定义，

**挑战者：** 有学位行者发心的定义。所以并不是完整发心的定义，就是他不周遍

<span style="color:blue">被挑战者： 字面上所讲的不是发心的定义。

**挑战者：** 那如果是这样，所以佛陀心中他的求菩提是怎么样求菩提法呢？

<span style="color:blue">被挑战者： 不求菩提，但是他还是在发心的定义当中。

**挑战者：** 好，所以总而言之，就是佛陀的心中仍然是有发心的，但是并不具备自立以及利他两种欲求，是这样子吗？

<span style="color:blue">被挑战者： 是

**挑战者：** 既然我们已经探讨完了佛陀的心中有没有菩提心的问题之后，我们进一步来探讨说佛陀的心中有没有归依呢？

<span style="color:blue">被挑战者： 知三宝的功德差别，自摄受等所含摄的心王和心所。
